# 顶级域名 (有时候也称1级域)
顶级域名通常是指如`wanwang.aliyun.com` 中的 `com`，是最后一个，最右边的一个，常用的有
 - top 适用个人网站，寓意顶尖的
 - com 经典域名
 - cn 中国的简称，常用于在中国服务的网站
 - org 适用于非营利组织、行业协会、国际组织

# 二级域名 (有人口语会说一级域名，但标准是这样)
二级域名通常就是如`hello.com`中的`hello`，这个就是从右往左第二个，顶级域名只有大企业或国家会拥有，二级域名是常用的
# 其他级别域名
其他级别的域名就是从右往左依次累加
# 协议层
比如有的时候会在域名前面加个`https://`，这就是协议层，像http/https协议域名后面还能接其他的路径，url参数
# 端口层
有的时候，域名后面会跟个`:111`之类的 : 数字的组合，这就是端口，因为域名其实就是代表了一个ip(经过dns查询得到)，这个端口就是指定访问这个ip的这个端口
# 示例
`https://cc.aaa.com:39999`
 - 顶级域名 com
 - 二级域名 aaa
 - 三级域名 cc
 - 协议 https
 - 端口 39999
